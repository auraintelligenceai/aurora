# `aura_intelligence pairing`

Approve or inspect DM pairing requests (for channels that support pairing).

Related:
- Pairing flow: [Pairing](/start/pairing)

## Commands

```bash
aura_intelligence pairing list whatsapp
aura_intelligence pairing approve whatsapp <code> --notify
```
